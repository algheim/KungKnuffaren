# Kungknuffaren

Chess bot written in C. Frontend in python using pygame. Far from done.

Inspired by Sebastian Lague: https://www.youtube.com/watch?v=U4ogK0MIzqk

## Running the Backend

1. `cd backend`
2. `make`
3. `bin/kungknuffaren`


## Running the frontend

1. `pip install -r requirements.txt`
2. `cd backend`
3. `make frontend_lib`
4. `cd ../frontend`
5. `python3 main.py`
